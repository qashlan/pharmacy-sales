================================================================================
         PHARMACY SALES ANALYTICS - Quick Start Guide
================================================================================

Thank you for using Pharmacy Sales Analytics!

This is a standalone application that requires NO installation.
Just double-click the executable to start!

================================================================================
  QUICK START
================================================================================

1. Double-click: PharmacySalesAnalytics.exe

2. Wait 5-10 seconds for the application to start

3. Your web browser will open automatically with the dashboard

4. Start analyzing your pharmacy sales data!

5. To close: Close the black console window or press Ctrl+C

================================================================================
  SYSTEM REQUIREMENTS
================================================================================

- Windows 10 or Windows 11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 1GB free disk space
- Any modern web browser (Chrome, Firefox, Edge)
- Internet connection (only for AI features)

================================================================================
  FIRST TIME USE
================================================================================

On first run, Windows might show a security warning:
- Click "More info"
- Then click "Run anyway"

This is normal for applications downloaded from the internet.
The app is safe and runs entirely on your computer.

================================================================================
  LOADING YOUR DATA
================================================================================

The application needs Excel files with your pharmacy sales data.

Required columns in your Excel file:
- Item Code
- Item Name
- Date (YYYY-MM-DD format)
- Selling Price
- Units (or Pieces, or Quantity)

Optional columns:
- Customer Name
- Category
- Time
- Receipt

To upload data:
1. Click "Upload Sales Data" in the sidebar
2. Browse and select your Excel file
3. Wait for it to load
4. Start exploring!

================================================================================
  TROUBLESHOOTING
================================================================================

Q: Application won't start?
A: Try running as Administrator (right-click > Run as administrator)

Q: Browser doesn't open automatically?
A: Look for the URL in the console window (usually http://localhost:8501)
   Copy and paste it into your browser

Q: Error loading data?
A: Make sure your Excel file has the required columns
   Try with sample data first (click "Use Sample Data")

Q: Application is slow?
A: Large datasets take time to process
   Close other applications to free up memory

Q: Port already in use error?
A: Close any other running instances of the application
   If problem persists, restart your computer
